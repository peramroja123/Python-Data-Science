#Rules to create a variable:

    Ex:- _9a=100
Rule 2 : Variable should be single word or a single character but not multiple words if you want please use underscore.
    Ex:- sai_kumar=800
Rule 3 : Python variables are commonly case sensitive 
    Ex:- city='hyd
         print(city)
